# Carousel Skill

Template para transformar um `SocialPost` de `instagram` com `type = "carrossel"` em uma sequência simples de slides.

## Estrutura padrão de 4 slides

1. Gancho
   Texto curto, direto e fácil de entender no mobile.

2. Dor
   Apresentar uma única dor forte e concreta ligada ao processo depois da oportunidade.

3. Processo/Solução
   Mostrar organização, clareza ou próximo passo de forma consultiva, sem parecer venda.

4. CTA
   Fechar com pergunta + ação simples.

## Regras

- 1 ideia por slide
- texto curto, mobile first
- linguagem consultiva
- evitar hype
- manter foco no "depois da oportunidade"

## Campos de saída esperados

- `slide1`: string
- `slide2`: string
- `slide3`: string
- `slide4`: string
